# wagunasu812's VPM Repository

VRChatアバター向けツールのVPMリポジトリです。

## VCCへの追加方法

1. VCC（VRChat Creator Companion）を開く
2. `Settings > Packages > Add Repository` を選択
3. 以下のURLを入力して追加：

```
https://wagunasu812.github.io/vpm-listing/index.json
```
